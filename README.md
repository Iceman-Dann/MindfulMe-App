# MindfulMe: Authentic Mental Wellness Platform

## Overview
A practical mental wellness application built with real technology to support users' mental health journey. This platform provides evidence-based tools and genuine AI-powered support through Cohere's language model API.

## Features

### Real AI-Powered Support
- **AI Wellness Counselor**: Powered by Cohere API for genuine conversational support
- **Evidence-based responses**: Trained to provide helpful mental health guidance
- **Crisis awareness**: Recognizes when professional help is needed
- **AI Personality Analysis**: Personalized insights based on user behavior patterns
- **AI Mood Prediction**: Predictive mood forecasting with recommendations
- **AI Personal Recommendations**: Customized wellness suggestions

### Practical Wellness Tools
- **Mood Tracking**: Daily mood logging with visualization
- **Journaling**: Private space for self-reflection
- **Goal Setting**: Personalized wellness goals and progress tracking
- **Habit Tracker**: Build positive mental health habits
- **CBT Exercises**: Cognitive Behavioral Therapy techniques
- **Enhanced Guided Meditation**: Premium mindfulness practices with advanced features
- **Coping Strategies**: Evidence-based stress management
- **Creative Expression**: Therapeutic drawing and art
- **Support Groups**: Peer community connection

### Enhanced Meditation Features
- **6 Meditation Sessions**: Morning Mindfulness, Stress Relief, Deep Sleep, Anxiety Relief, Focus Enhancement, Breathing Exercise
- **Interactive Audio Controls**: Play, pause, stop, volume, and seek functionality
- **Custom Meditation Timer**: Set personalized meditation duration (1-60 minutes)
- **Breathing Exercise Guide**: Animated breathing patterns with visual feedback
- **Fullscreen Mode**: Immersive meditation experience
- **Progress Tracking**: Real-time audio progress visualization
- **Session Categories**: Organized by mood, time, and difficulty level
- **Beautiful UI**: Modern, animated interface with gradient backgrounds

### User Experience Improvements
- **Profile Logout**: Complete data reset for demo purposes
- **Enhanced UI/UX**: Modern design with smooth animations
- **Responsive Design**: Optimized for all device sizes
- **Dark Mode**: Eye-friendly interface option

- Empathetic conversational support

## Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Iceman-Dann/MindfulMe-App/
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npm start
   ```

## Code Overview

The app is structured with various components:

- **`App.js`**: Handles themes, routes, and authentication.
- **`store`**: Redux store and slices for state management.
- Sub-folders in **`components`**: Different features like Mood Tracker, Peer Chat, Goal Planner, etc.
- **`utils`**: Helper functions, Firebase setup, and more.
